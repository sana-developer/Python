Databases & SQL Basics
A database is a structured way to store and manage data. SQL (Structured Query Language) is used to interact with relational databases like PostgreSQL, MySQL, and SQLite.

1️⃣ Common SQL Commands
Command	Purpose
CREATE TABLE    -	Creates a new table
INSERT INTO     -	Adds data to a table
SELECT  -	Retrieves data
UPDATE	-   Modifies existing data
DELETE	-   Removes data
